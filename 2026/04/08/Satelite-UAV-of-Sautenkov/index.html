Mission Planing For UAV by Oleg Sautenkov

此笔记记录Oleg Sautenkov三篇关于UAV-Mission Planing的文章。

Mission Planing

为无人机设计一条从起点到终点、满足任务要求且最优的飞行路径与动作序列。

  • 路径与动作：不仅规划“怎么飞”（路径），还规划“做什么”（动作，如在何处悬停、拍照、传输数据。
  • 满足约束：必须遵守任务要求（如避开禁飞区、保持与卫星的通信、在电池续航内完成）。
  • 追求最优：在众多可行方案中，找到能耗最少、时间最短或信号最好的平衡解

UAV-VLA: Vision-Language-Action System for Large Scale Aerial Mission Generation

主要针对利用卫星图（俯瞰）+ 高层指令输出航点序列（经纬度）

Problem

  • 传统UAV任务规划严重依赖专业操作员，使用地面站软件（如Mission Planner）手动设置航点。
  • 现有AI方法（VLA/VLN）的局限
    • 数据饥渴： 需要大量“语言-动作”配对数据进行训练，在真实、多样的UAV场景中难以获得。
    • 缺乏全局视野： 许多方法专注于第一视角的导航（VLN），缺乏从“上帝视角”（卫星图）进行全局、高效的路径规划。

Innovation

  • 提出了一个大规模 VLA 系统，输入是“基于文本的任务请求 + 卫星图像”，输出是“路径-动作集”。
  • 发布了 UAV-VLPA-nano-30 基准数据集，用于在全球尺度下评估 VLA 系统。
  • 在 UAV-VLPA-nano-30 上验证了系统，结果显示其性能可与人类水平的路径和动作生成相媲美。

Method

UAV-VLPA-nano-30：从美国地质调查局（USGS）的公开平台 EarthExplorer 收集的 30张高分辨率卫星图像。图像覆盖美国各地，包含多种环境，以确保评估的全面性。

  • 关键元数据：每张图都带有地理描述信息（经纬度范围等），用于将图像上的像素点转换为真实的经纬度坐标，以便生成无人机可以执行的飞行计划
    • 分辨率：约 1.5 米/像素，能清晰看到地面特征。
    • 覆盖面积：每张图覆盖约 760 平方米。
    • 拍摄条件：春/夏季的白天，确保光线良好、视野清晰。
  • 人类专家用Mission Planner手动生成的飞行计划。

UAV-VLPA*: Vision-Language Mission Planning with Global-Local Optimization for Scalable UAV Route Generation

Problem

针对UAV-VLN 结果按 VLM 输出的顺序访问（从左到右、从上到下），不考虑障碍物，路径效率低

Innovation

  • 利用TSP进行全局优化，利用A*算法进行局部避障
  • 效果超越人类

Method

Instruction parsing, Vision-Languag eProcessing,and Waypoint Generation 相比UAV-VLN增加目标-障碍物（需要避开的，如“湖泊”）同时解释文本中的限制条件（如“避开湖泊”）

  • 输出：航点列表（waypoints）+禁飞区列表（prohibited regions）

Traveling Salesman Problem 使用2-opt 局部搜索重新排序，找最短路径

初始顺序：A → B → C → D → E 尝试交换两条边： A → B → C → D → E (原始) A → C → B → D → E (交换后) ↓ 如果新路径更短 → 保留 继续尝试，直到无法改进

Path Planning and Obstacle Avoidance

  • 将地图离散化为由大小相等的网格单元组成的统一网格，每个单元表示图中的一个节点（每个网格代表5x5的像素）
  • VLM（视觉语言模型）会识别出障碍物（如湖泊、建筑）。如果一个格子与任何障碍物有重叠，它就会被标记为“不可通行”。
  • 所有与障碍物（VLM 标记的湖泊等）相交的格子，都会被标记为“障碍格”，A* 在搜索路径时，会完全忽略这些“障碍格”，就像它们不存在一样。这样，算法规划的路径天然就会绕过这些格子，从而实现避障。

效果 人类 62.95 km 基准 纯 A*（无 TSP） 76.84 km +22.1% ❌ 纯 TSP（无 A*） 51.34 km -18.5% ✅ TSP + A* 51.27 km -18.5% ✅
